## Project description

Bike Store

## Who uses it?

Customers who want to buy bikes.

## Outputs from MO Bikes Store Web App

Outputs for users

- Browse Bikes by Year
- Browse Bikes by Make
- Browse Bikes by Body Type
- Browse Bikes by Price Range
- Schedule Test Drive

Outputs for Admin

- View Inventory
- Add Inventory
- Edit Inventory
- Delete Inventory
- Followup with potential Bikes buyers
